Documentação completa do sistema de missões implementado no projeto.

## Ferramentas ExtraTools Relacionadas

O Quest System possui ferramentas de criação e edição disponíveis no menu `Extra Tools/Quest System/`:

- **Quest Creation Tool** - Ferramenta para criar novas missões
- **Quest Sprite Generator** - Gerador de sprites para missões

Ver documentação em [../Tools/ExtraTools/](../Tools/ExtraTools/README.md) para mais detalhes sobre as ferramentas de desenvolvimento.

## Estrutura do Sistema

O Quest System está organizado em:

- **Core System** - Lógica principal das missões
- **UI System** - Interface de usuário para missões
- **Tools** - Ferramentas de criação e edição (ExtraTools)
- **Testing** - Cenários e ferramentas de teste
